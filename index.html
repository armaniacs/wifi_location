Wifi location

Get your location with WiFi Mac Address and GoogleMap.

Download Download View Wifi location on GitHub

REQUIREMENTS:

  • Ruby 1.8.7+ or 1.9.2+
  • WiFi
  • Mac OSX or Linux

INSTALL:

% gem install wifi_location

HELP:

% whereami --help
options:
 -help (-h)     show help
 -towers        dump WiFi towers
 -dump          dump ALL
 -map           show Google Map URL
 -version (-v)  show version
--
wifi_location v0.0.6 - http://shokai.github.com/wifi_location

GET Location:

% whereami
lat=35.3880376,lon=139.4262089
% open `whereami --map`
=> https://maps.google.com/?ll=35.3880376,139.4262089
